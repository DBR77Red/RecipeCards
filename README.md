# RecipeCards

A card-based recipe project for organizing and displaying recipes in a clean card layout.

## Project Structure

```
RecipeCards/
├── README.md
├── index.html
├── src/
│   ├── styles/
│   └── scripts/
└── assets/
    └── images/
```

## Getting Started

Open `index.html` in your browser to view the recipe cards.
